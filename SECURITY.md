# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Use [GitHub Security Advisories](https://github.com/sert-xx/unified-blueprint/security/advisories/new) to report privately
3. Or email: demo.sert@gmail.com

## Response Process

1. Acknowledgment within 48 hours
2. Investigation and fix development
3. Security advisory publication
4. Patch release

Thank you for helping keep UBP secure.
